Parágrafos e Quebras de linha

Para criar um parágrafo, basta escrever qualquer texto entre os símbolos <p> e </p>.

Você pode até criar outro parágrafo que nem esse, mas não importa quantas vezes você use a tecla "enter" para quebrar o texto, o que você escreveu continuará no mesmo parágrafo.

Para isso existe o comando "br".
Ele faz essa quebra de linha para você (Acabei de usar nesse).
