Comandos usados:

var num1 = Number.parseInt(window.prompt ("Digite um número: "))
var num2 = Number.parseInt(window.prompt ("Digite outro número: "))
var soma = num1 + num2
window.alert ("A soma dos números é: " + soma)

(Esse parseInt, também existe o parseFloat, converte um valor string para um valor int. Isso porque o comando var num1 = Number.parseInt(window.prompt ("Digite um número: ")) converte o dado recebido em string.)
